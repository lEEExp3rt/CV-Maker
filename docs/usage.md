# Usage Guide

本文档为使用指南。

## Project Sidebar

左侧侧边栏管理多个简历项目，所有数据自动保存到浏览器 localStorage。

| 操作 | 说明 |
|------|------|
| 点击项目名 | 切换到该项目 |
| **+ 新建项目** | 创建空白项目 |
| **⧉** | 复制当前项目（含全部数据） |
| **✎** | 重命名项目 |
| **×** | 删除项目（至少保留一个） |
| **◀** | 收起侧边栏为窄条 |
| 拖动右侧边界 | 调整侧边栏宽度 |

## Form Tabs

六个选项卡，对应简历的六个模块。

### Personal Information

填写姓名和联系方式。照片和自定义链接为可选项。

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓名 | 是 | 简历标题 |
| 照片 | 否 | 上传本地图片，base64 存储。有照片显示带照片布局，无照片显示居中布局 |
| 邮箱 | 是 | mailto 链接，可点击 |
| 电话 | 是 | 纯文本展示 |
| 个人主页 | 否 | 展示 URL，可点击 |
| GitHub | 否 | 展示 URL，可点击 |
| 自定义链接 | 否 | 见下方说明 |

**自定义链接**：每条链接可设图标（5 种预设 + URL 输入 + 上传），URL 以 `http://` 或 `https://` 开头自动加超链接，否则显示为纯文本。

### Educations

| 字段 | 必填 | 说明 |
|------|------|------|
| 学校 | 是 | 中文名 |
| 英文名 | 否 | 斜体展示 |
| 学位 | 是 | 如"本科"、"硕士" |
| 专业 | 是 | - |
| 开始 | 是 | 如 `2022-09` |
| 结束 | 否 | 留空显示"至今" |
| GPA | 否 | - |
| 排名 | 否 | - |
| 核心课程 | 否 | 逐条添加，每门课一个输入框 |

### Internships

| 字段 | 必填 | 说明 |
|------|------|------|
| 公司 | 是 | 中文名 |
| 英文名 | 否 | 斜体展示 |
| 部门 | 否 | 与岗位用 `·` 连接 |
| 岗位 | 是 | - |
| 开始 | 是 | - |
| 结束 | 否 | 留空显示"至今" |
| 摘要 | 否 | 支持 Markdown |
| 详细描述 | 是 | 逐条添加，支持 Markdown |

### Projects

| 字段 | 必填 | 说明 |
|------|------|------|
| 项目名 | 是 | 中文名 |
| 英文名 | 否 | 斜体展示 |
| 项目链接 | 否 | 展示 URL，可点击 |
| 开始 | 是 | - |
| 结束 | 否 | 留空显示"至今" |
| 摘要 | 否 | 支持 Markdown |
| 详细描述 | 是 | 逐条添加，支持 Markdown |

### Skills

逐条添加技能分类（如"编程语言"），每个分类下逐条添加具体技能。整节可选，留空不显示。

### Awards

每条包含奖项名称和日期。整节可选。

## Settings Bar

顶部工具栏下方有两个设置项，修改即时生效：

| 设置 | 选项 |
|------|------|
| 配色方案 | 深蓝 / 石板灰 / 森林绿 / 勃艮第红 / 青墨 / 炭黑 |
| 语言模式 | 中文（中英双语标题）/ English（纯英文标题） |

## Toolbar Buttons

| 按钮 | 功能 |
|------|------|
| 导出 JSON | 弹窗展示格式化 JSON，可 📋 复制或下载文件 |
| 导入 JSON | 粘贴 JSON 代码或选择本地文件导入 |
| 清空 | 重置当前项目（有确认弹窗） |
| 打印 PDF | 浏览器打印 → 另存为 PDF（A4），仅输出简历部分 |

## Markdown Support

`摘要` 和 `详细描述` 字段支持以下行内格式：

| 语法 | 效果 |
|------|------|
| `**文字**` | **粗体** |
| `*文字*` | *斜体* |
| `__文字__` | _下划线_ |
| `` `代码` `` | `等宽字体` |

## Icons

**内置图标**：使用 [Remix Icon](https://remixicon.com)（MIT 协议），通过 `remixicon` npm 包提供。

**自定义链接图标**：每条自定义链接可单独设置图标：

| 方式 | 说明 |
|------|------|
| 预设 | 20+ 内置预设（邮箱、电话、网站、GitHub、微信、QQ、知乎、B 站等） |
| Remix 图标名 | 直接输入 Remix Icon 的图标名（如 `discord-fill`、`twitter-fill`），自动从 2800+ 图标中加载 |
| URL | 远程图标地址或本地路径（如 `icons/xxx.svg` 或 `https://...`） |
| 上传 | 选择本地图片，base64 存储 |

不设图标则使用默认链接图标。在 [Remix Icon 官网](https://remixicon.com) 搜索图标名，复制名称填入编辑器的「自定义链接 → 图标 → Remix」输入框即可。
